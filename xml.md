Assignment #2 
(INSY 8414 - Mobile Programming) 
Academic year: 2025-2026 
Please submit on time! 
What to do: 
Create an Android application with three screens (Activities): 
1. 
Login Screen 
2. Sign-Up Screen 
3. User Details Screen 
Users should be able to navigate between the three screens using buttons or links. 
Requirements: 
● Each screen must use a variety of Layouts, Views, and Widgets (e.g., 
LinearLayout, TextView, EditText, Button, RadioButton, CheckBox). 
● Add buttons/links to move from: 
○ Login → Sign-Up 
○ Sign-Up → User Details 
○ Login (successful) → User Details 
Extra Credit (Optional) 
● Add simple validation (e.g., empty fields not allowed). 
● Show a Toast message confirming navigation (e.g., “Login successful”)
